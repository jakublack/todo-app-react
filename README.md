# Simple TODO-app

###Run on Netlify:

https://quirky-easley-63fe26.netlify.com


### Installing

clone repository

```
git clone https://github.com/jakublack/todo-app-react.git
```

install project dependencies

```
npm install
```

### Development

Start project in dev mode

```
npm run start
```

### Production
Build production pack

```
npm run build
```
